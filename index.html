HTML Table Reference

Table Tags

Tag Name Description
<table> Table The tag initiates the table in html code normally followed by <thead>
<thead> Table Head The set of rows define the column header in a table.
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row container.
<th> Table header cell The table data define the header of row or column.
<td> Table Data The row container.
<tfoot> Table Foot The set of rows define the footer in a table.

Display: flex/inline-flex

code values description
justify-content flex-start, flex-end, center, space-between, space-around defines how the browser distributes space between and around content items along the main-axis of their container.
flex-direction row, row-reverse, column, column-reverse specifies how flex items are placed in the flex container - either vertically or horizontally.
flex-wrap nowrap, wrap, wrap-reverse specifies whether flex items should wrap or not.
flex-flow value {flex-direction flex-wrap} shorthand for flex-direction and flex-wrap
align-content start, end, flex-start, flex-end, center, space-between, space-around, space-evenly, stretch modifies the behavior of the flex-wrap property. It determines how to space rows from top to bottom (ie. along the cross axis). Multiple rows of items are needed for this property to take effect.
flex-grow/shrink number, grow's default is "0", shrink's default is "1" specifies how a flex item will grow or shrink so as to fit within the space available in its flex container.
flex-basis size(px?), auto sets the initial base size for a flex item before any other space is distributed according to other flex properties.
flex value {flex-glow flex-shrink flex-basis} shorthand for flex-glow/shrink and basis
