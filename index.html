HTML Table Reference

Table Tags

Tag Name Description
<table> Table The tag initiates the table in html code normally followed by <thead>
<thead> Table Head The set of rows define the column header in a table.
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row container.
<th> Table header cell The table data define the header of row or column.
<td> Table Data The row container.
<tfoot> Table Foot The set of rows define the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many column length the cell will have
rowspan Row Span Defines how many row length the cell will have
